# TSenhanced

## Compile

#### Compile files in directory to TypeScript

```bash
    $ ./main.py -d [directory]
```

#### Compile files in directory to JavaScript (not implemented)

```bash
$ ./main.py -j [directory]
```

### Compile and execute with Deno

#### Single File

```bash
$ ./main.py -f [file]
```

#### All files in directory

```bash
$ ./main.py -a [directory]
```

Default file is `index.tsa`
